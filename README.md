# Buildo Know Mini App

> AI-поиск по всем документам компании. Ответ за 3 секунды

Telegram Mini App (открывается через Telegram-бот).

## Локальная разработка

```bash
npm install
npm run dev
# → http://localhost:5173
```

## Build

```bash
npm run build
# → dist/ (committed to repo for Layero static delivery)
```

## Деплой

Платформа: **Layero** (static, dist/ precommitted).

## Стек

- Vite 6 + React 19
- TypeScript 5.4
- Tailwind CSS 3.4
- Brand color: `#10B981`
